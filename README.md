## Описание

Здесь я пошагово выполняю задания с курса по автоматизации: от базового Java-кода до написания автотестов для веб-интерфейса и API.  
Репозиторий нужен как учебное портфолио и база примеров для подготовки к собеседованиям.

## Технологии

- Язык: Java
- Сборщик: Maven
- Тестовый фреймворк: JUnit / TestNG
- Автотесты: Selenium / RestAssured / другие (по мере появления)
- Отчётность: Allure
- IDE: IntelliJ IDEA

## Структура проекта

- `src/main/java` — учебный код
- `src/test/java` — тесты по разделам курса
- `src/test/resources` — тестовые данные, конфиги
- `pom.xml`

(Эту секцию возможно буду обновлять по мере того, как курс добавляет новые модули.)
